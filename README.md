# Telecom-Customer-Churn-Prediction

Project Title: Telecom Customer Churn Prediction & Risk Analysis.

Business Problem: The telecommunications company faces customer attrition. The goal is to proactively identify high-risk customers using historical data and provide actionable insights to the retention team.

Tech Stack: Python, Pandas, Scikit-Learn, SMOTE, Power BI.

Machine Learning Approach:

    Handled a highly imbalanced dataset using SMOTE (Synthetic Minority Over-sampling Technique).

    Trained a Random Forest Classifier that achieved an overall accuracy of 92% and significantly improved the recall for the churn class.
